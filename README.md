# 🐧 PenGuard: AI Grammar & Spell Checker

PenGuard is a lightweight, AI-powered grammar and spell-checking web application designed to help users refine their writing with precision and ease. Utilizing an interactive web interface backed by a robust backend controller, PenGuard analyzes text inputs and provides real-time corrections.

---

## 🚀 Features

* **AI-Powered Corrections:** Detects subtle grammatical errors, structural typos, and spelling mistakes.
* **Real-Time Processing:** Fast, responsive text evaluation directly from the web interface.
* **Clean UI:** Minimalist and modern front-end designed with accessible Google Fonts (`DM Serif` & `DM Mono`).
* **Modular Architecture:** Structured with a dedicated C# backend controller for high reliability and scalability.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Custom variables, responsive layout)
* **Backend:** C# (.NET Core)
* **Fonts:** Google Fonts API
